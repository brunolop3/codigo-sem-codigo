# Task 1.5 — Conectar Sheets Component

## Summary
Created `/home/z/my-project/src/components/sections/conectar-sheets.tsx` — an interactive step-by-step tutorial showing how to connect a web tool to Google Sheets via Apps Script.

## What was built
- **Section** with `id="conectar"`, lime-colored "Conexão Essencial" badge with Plug icon, title "Conectando ao Google Sheets" (with "Google Sheets" in lime), and descriptive subtitle
- **6-step interactive stepper** with:
  1. **Criar a Planilha** — Mock Google Sheets with headers (Protocolo, Nome, Setor, Data, Status) + tip callout
  2. **Abrir o Apps Script** — Mock browser showing Extensões > Apps Script menu + mock Apps Script editor + error callout
  3. **Colar o Código** — Real, working, commented Apps Script code with syntax highlighting, copy button, toast notification + warning callout
  4. **Implantar como App da Web** — Mock deploy dialog with proper configuration (Type: App da Web, Executar como: Eu, Acesso: Qualquer pessoa) + error callout about re-deploying
  5. **Configurar Permissões** — Two mock Google permission screens (authorization required + unverified app) + tip callout
  6. **Colar a URL na Ferramenta** — Mock URL display, tool configuration mockup, success animation with spring checkmark + tip callout

## Visual Design
- Progress bar with step indicators (circles connected by lines), pulse animation on current step
- Filled circles for completed steps (with check), numbered for current, locked for future
- AnimatePresence for smooth step transitions
- Previous/Next navigation buttons at bottom with dot indicators
- Clickable completed steps for revisiting
- Mobile responsive
- Dark theme: bg-surface, text-foreground, text-muted-lavender, lime/coral accents
- Three callout types: Tip (lime), Error (coral), Warning (yellow)

## Technical Details
- `'use client'` directive
- framer-motion for step transitions, pulse animations, and spring checkmark
- shadcn/ui: Button, Card, CardContent, Badge
- sonner toast for copy feedback
- lucide-react icons
- All text in Brazilian Portuguese
- Lint passes clean (fixed JSX comment and object literal escaping issues)
