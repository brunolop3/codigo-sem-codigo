# Task 2.5 — Comparador Prompt Component

## Summary
Created `/home/z/my-project/src/components/sections/comparador-prompt.tsx` — an interactive Before/After prompt comparator component.

## What was built
- **Section** with `id="comparador"` — Badge "Antes e Depois" with ArrowRightLeft icon (coral), title "Veja a Diferença" (Diferença in lime), subtitle in Portuguese
- **Toggle Switch** — switches between "Antes (Prompt Ruim)" and "Depois (Prompt Bom)" with coral/lime accent colors and smooth spring animation via framer-motion
- **Bad Prompt view** — shows the vague prompt with strikethrough styling, issues footer listing problems
- **Good Prompt view** — shows the detailed prompt with lime-highlighted diff lines (specific features, quantities, Google Sheets mention, ask-to-ask instruction, single file instruction)
- **Annotation cards** — 5 cards on the side (below on mobile) explaining why each improvement matters, with icons and descriptions
- **Bottom CTA button** — toggle between views
- **Dark theme** throughout, matching the project's existing design system (navy, lime, coral colors)
- Used framer-motion AnimatePresence for smooth transitions between views
- Used shadcn/ui Badge, Card, CardContent, Button components
- Used lucide-react icons (ArrowRightLeft, ThumbsDown, ThumbsUp, X, Check, Lightbulb)
- All text in Brazilian Portuguese
- 'use client' directive
- Exported as default function ComparadorPrompt
